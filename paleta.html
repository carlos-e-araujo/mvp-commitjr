Paleta de Cores Vibrantes e Neutras

Explore esta seleção de cores cuidadosamente escolhida para oferecer uma base sólida e versátil para o seu design. A paleta combina tons de azul e verde vibrantes com neutros clássicos.

Cores Principais

Azul Vibrante

Ideal para elementos de destaque, links e botões de chamada para ação (call-to-action).

#0080F0

Verde-azulado (Teal)

Perfeito para seções informativas ou como uma cor de destaque mais sóbria.

#008080

Ciano Brilhante

Excelente para ícones, alertas ou para adicionar um toque de modernidade e frescor.

#06B6D4

Cinza Escuro (Charcoal)

Ideal para textos principais pela sua alta legibilidade e para fundos escuros.

#333333

Cores Personalizadas / Neutras

Branco Acinzentado

Perfeito para fundos de página, cartões e seções, sendo uma alternativa suave ao branco puro.

#F8F9FA

Branco Puro

Essencial para criar espaços em branco, garantir contraste e uma aparência limpa.

#FFFFFF

Cinza Médio

Ótimo para textos secundários, legendas, ícones menos proeminentes e bordas.

#555555
